## Basic Drag Feature

<Horizontal>
<div style={{maxWidth: '70%'}}>
You can drag images to a `<DragDropContentView />` component or any other app that supports drag and drop.
```tsx
export const IDragDropContentView = (props) => {
  const [images, setImages] = useState<OnDropEvent[] | null>(null);
  const sources = images?.map((image) => image.uri || image.base64);

  return (
    <DragDropContentView
      draggableImageSources={sources}
      style={styles.container}
    >
      <Image {...props} />
    </DragDropContentView>
  );
};
```
</div>

  <image href={useBaseUrl("img/drag.gif")} width="100%" height="100%" />
</svg>
</Horizontal>

### Web Specific Caution:

- If you're using `<Image />` component from `react-native`, you need to set `draggable=true`.`Expo Image` handles this automatically.

```tsx
import { Image } from "react-native";

<Image {...props} draggable={true} />;
```
